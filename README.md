# NVIDIA Preset Launcher

Simple launcher for selecting NVIDIA driver presets based on GPU generation.

## Presets

- **Preset M** → RTX 40 / 50 Series
- **Preset K** → RTX 20 / 30 Series

## Usage

1. Download `Nvidia_Preset_Launcher.bat`
2. Run as Administrator
3. Select your GPU generation
4. PowerShell will open and apply the preset

## Notes

- Scripts are downloaded directly from this repository.
- ExecutionPolicy is handled automatically by the launcher.
- No manual PowerShell configuration required.
